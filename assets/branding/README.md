# Cute Sticky branding assets

- `cute-sticky-icon-master.png` is the 1024×1024 RGBA master for desktop and PWA icons.
- `cute-sticky-tray.svg` is the deterministic monochrome source for the macOS menu-bar template icon.

The app icon was generated with the built-in image generation workflow, then limited to a single recognizable mark: a cream sticky note, folded corner, and peach check-heart on a warm orange tile. Generated desktop and web sizes live under `src-tauri/icons/` and `public/`.
